📋 SurveyJS 功能清單

問卷系統目前使用的 SurveyJS Creator 功能盤點（題目類型、編輯器設定、統計分析）

📝 支援的題目／答覆類型

統計分析模組目前識別並支援以下題型

顯示名稱 SurveyJS 類型 說明 統計支援
單選題 radiogroup 從多個選項中選擇一個（單選按鈕） ✓ 支援
下拉選單 dropdown 下拉式選單（單選），歸類為單選題統計 ✓ 支援
是／否題 boolean 布林開關，歸類為單選題統計（自動轉為「是／否」） ✓ 支援
多選題 checkbox 可勾選多個選項 ✓ 支援
單行文字 text 單行輸入框 ✓ 支援
多行文字 comment 多行文字輸入區（含長段文字） ✓ 支援
評分題 rating 數字評分（如 1-5 星、1-10 分） ✓ 支援
其他題型 matrix、file、imagepicker… 等 SurveyJS 內建仍可使用，但統計分析僅顯示「回答數量」 部分支援(需實作)
💡 上述題型由 SurveyJS Creator 工具箱（toolbox）預設提供。若需限制使用者僅能新增上述題型，可透過 Creator 的 toolbox 客製化進一步收斂。

🛠️ Survey Creator 其他內建功能

以下為 SurveyJS Creator 套件預設啟用的功能,使用者建立問卷時即可使用,無需額外開發。

編輯器分頁 (Tabs)

分頁 狀態 說明
設計 Designer ✓ 啟用 主要編輯介面,可從工具箱拖曳題目、調整順序、設定屬性
預覽 Preview ✓ 啟用 直接在編輯器內預覽問卷外觀與操作流程
邏輯 Logic ✓ 啟用 以圖形化方式管理條件規則(顯示/隱藏題目、必填條件、計算欄位)
翻譯 Translation ✓ 啟用 多語系翻譯介面,可為每題輸入不同語言版本
主題 Themes ✓ 啟用 調整問卷外觀(色彩、字型、圓角、陰影等視覺樣式)
JSON 編輯 ✗ 已關閉 原始 JSON 編輯分頁,本系統刻意隱藏避免使用者直接編輯資料

題目編輯功能

功能 說明
工具箱 Toolbox 左側拖曳區,可將題型拖入編輯區或點擊新增
屬性面板 Property Grid 右側面板,可細部調整每題的設定:標題、名稱、必填、預設值、版面寬度等
多頁問卷 Pages 可建立多頁問卷,題目可分組於不同頁面,填寫時逐頁顯示
題目群組 Panel 可將題目聚合成 Panel(子區塊),支援巢狀結構
動態題目 PanelDynamic 讓使用者自行新增多筆相同結構的資料(如多位家屬資料)
複製/刪除題目 每題右上選單可複製、刪除題目
拖曳排序 支援題目、頁面、選項皆可拖曳排序
撤銷 / 重做 Undo/Redo 編輯動作可復原(Ctrl/Cmd + Z)與重做

驗證與條件邏輯

功能 說明
必填驗證 可設定 isRequired,未填時無法送出
內建驗證器 數字範圍、文字長度、Email 格式、Regex 正規表示式、答案唯一等驗證規則
顯示條件 visibleIf 依其他題目答案決定本題是否顯示(分支邏輯)
啟用條件 enableIf 依條件控制題目是否可作答(唯讀切換)
必填條件 requiredIf 依條件決定是否變為必填
計算欄位 Calculated Values 可用 SurveyJS 表達式即時計算,例如 BMI、總分等衍生欄位
觸發器 Triggers 依條件自動執行動作(設定值、複製值、跳頁、完成問卷、執行 JS)

本系統已使用的 Creator 屬性設定

設定項 值 說明
autoSaveEnabled true 編輯時自動儲存(不需手動點選儲存按鈕)
collapseOnDrag true 拖曳題目時自動收合,使編輯區更整潔
showJSONEditorTab false 隱藏 JSON 編輯分頁,避免使用者直接修改原始資料
locale zh-tw 編輯器與問卷介面皆為繁體中文
themeForPreview defaultV2 預覽分頁使用 SurveyJS V2 主題
💡 上述「編輯器分頁」與「題目編輯功能」皆為 Creator 預設行為,可依需求透過 ICreatorOptions 關閉或客製化,例如:
  • showLogicTab: false 隱藏邏輯分頁
  • showTranslationTab: false 隱藏翻譯分頁
  • showThemeTab: false 隱藏主題分頁
  • questionTypes: [...] 限制工具箱可用題型

📈 統計分析功能

位於「查看回覆 → 統計分析」分頁

目前問卷系統已經設計的題型統計輸出

題型 統計內容 呈現方式
單選題 / 下拉 / 是否 每個選項的人數與百分比 選項依問卷原順序排列，顯示 N 人 (XX.X%)
多選題 每個被選擇的選項人數與百分比 依人數由高到低排序，未被選擇的選項不顯示
文字題（單行／多行） 所有去重後的回答 列出每一段獨特文字回應
評分題 平均分、最高分、最低分 平均分取小數點後一位，無回應時顯示「尚無評分」
其他題型 僅顯示回答數量 純計數，不分析內容

🗂️ 問卷管理功能

功能 位置 說明
新增問卷 首頁 → 新增問卷按鈕 進入空白編輯器，自動儲存後建立紀錄
編輯問卷 問卷卡片 → 編輯問卷 使用 SurveyCreator 元件，編輯時自動儲存
預覽問卷 問卷卡片 → 預覽 新分頁開啟，唯讀模式檢視填寫畫面
複製連結 問卷卡片 → 複製連結 產生匿名填寫連結，並提示無法追蹤個案身份
查看回覆 問卷卡片 → 查看回覆 進入回覆清單與統計分析兩個分頁
複製問卷副本 問卷卡片右上選單 以對話框輸入新名稱後建立副本
刪除問卷 問卷卡片右上選單 以對話框二次確認後刪除

📨 回覆相關功能

功能 位置 說明
問卷填寫 SurveyFillPage 使用 survey-react-ui 的 <Survey> 元件，完成時呼叫 submitResponse
回覆清單 查看回覆 → 回覆清單分頁 表格顯示填寫時間、姓名、身分證字號；支援搜尋姓名與時間排序
回覆詳情 清單列 → 查看詳情 新分頁開啟，以 mode: "display" 唯讀顯示原問卷與填寫結果
統計分析 查看回覆 → 統計分析分頁 逐題顯示統計摘要（見上方統計分析章節）
